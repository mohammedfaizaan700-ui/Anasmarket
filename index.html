Bitcoin Price Prediction System

Project Description

This project analyzes historical Bitcoin prices and predicts future prices using Machine Learning (Linear Regression).

Technologies Used

  • Python
  • Pandas
  • NumPy
  • Matplotlib
  • Scikit-Learn

Features

  • Historical Price Visualization
  • Price Prediction using Linear Regression
  • Actual vs Predicted Price Comparison
  • Mean Squared Error Calculation
